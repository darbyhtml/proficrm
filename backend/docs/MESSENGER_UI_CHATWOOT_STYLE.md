# Настройки Messenger в стиле Chatwoot

## Цель
Упорядочить UI/UX настроек мессенджера по образцу Chatwoot: отдельная страница «Источники», пошаговое добавление источника, вкладки при редактировании, превью виджета.

## Изменения

### 1. Страница «Источники» (список)
- Заголовок «Источники», краткое описание (канал = способ связи, входящий = управление диалогами).
- Ссылка «Узнать больше о Входящих» (опционально).
- Кнопка «Добавить источник».
- Список источников: иконка по типу (сайт / API), название, тип («Сайт», «Источник API»), действия — шестерёнка (настройки), корзина (удалить).

### 2. Добавление источника (пошагово)
- **Шаг 1 — Выберите источник:** сетка карточек (Сайт, API, остальные «Скоро»).
- **Шаг 2 — Создать источник:** форма по типу (для Сайт: имя, домен, цвет, приветствие).
- **Шаг 3 — Готово:** «Источник готов!», код вставки, «Копировать», «Перейти к настройкам», «Больше параметров».

### 3. Редактирование источника
- Вкладки: «Конструктор виджета» (внешний вид), «Настройки» (имя, филиал, токен, код), «Время работы», «Безопасность», «Интеграции» и т.д.
- Справа: переключатель «Предпросмотр» / «Скрипт», живое превью виджета (iframe или заглушка).

### 4. Навигация
- В левом сайдбаре: «Настройки» → «Источники» (текущая страница списка).
- Хлебные крошки: «Настройки» / «Источники» / «Добавить источник» или «Название источника».
