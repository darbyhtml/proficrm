Справка по Widget API

Эндпоинты

  • POST /api/widget/bootstrap/ — создание/получение сессии
  • POST /api/widget/send/ — отправка сообщения
  • GET /api/widget/poll/ — получение новых сообщений
  • GET /api/widget/stream/ — SSE-поток обновлений

Особенности

  • Автоматическое создание сессии при первом открытии
  • SSE-стриминг для мгновенных обновлений (fallback на polling каждые 3 сек)
  • Автоматический re-bootstrap при истечении сессии (401)
  • Локальное хранение session_token и since_id в localStorage
