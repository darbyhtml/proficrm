{% comment %} v2-modal + v2-toast — лёгкий SPA-слой без внешних библиотек. API: window.V2Modal.open({ url, title, onSaved, onClose }) — загружает partial HTML и показывает в модалке window.V2Modal.openHtml({ html, title, onClose }) — показывает готовый HTML window.V2Modal.close() — закрывает текущую модалку (с проверкой dirty) window.V2Toast.show(msg, {type:'ok'|'err'|'info', timeout:3000}) События на window: 'v2-modal:saved' (detail = {url, response}) — dispatch из partial после успешного сохранения Поведение: - Esc / клик по backdrop → close (с confirm, если форма dirty) - Форма внутри модалки перехватывается: submit → fetch POST, обрабатывает JSON {ok:true, toast, close:true} или partial HTML (перерисовывает контент модалки — валидационные ошибки). {% endcomment %}
