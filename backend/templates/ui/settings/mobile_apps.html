Как работает auto-update

CRMProfiDialer периодически вызывает GET /api/phone/app/latest/ — возвращает самую свежую активную production-версию (сортировка по version_code).

Если приложение получает версию с кодом больше своего — предлагает пользователю скачать и установить. SHA256 используется для проверки целостности загруженного файла.

Только одна активная версия с наивысшим version_code «видна» из приложения. Отключённые (is_active=False) остаются в истории, но не предлагаются для обновления.
