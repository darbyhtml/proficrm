📋 Пошаговая инструкция по настройке OAuth

Шаг 1: Создание интеграции в AmoCRM

  1. Войдите в ваш аккаунт AmoCRM: {{ cfg.domain|default:"kmrprofi.amocrm.ru" }}
  2. Нажмите на три точки (⋯) в правом верхнем углу
  3. Выберите "amoMarket" или перейдите в "Настройки" → "Интеграции"
  4. Нажмите "Создать интеграцию" или "Добавить интеграцию"
  5. Заполните форму:
    • Название: любое (например, "CRM Импорт")
    • Описание: опционально
  6. После создания интеграции AmoCRM покажет вам Client ID и Secret key

Шаг 2: Настройка Redirect URI

  1. В настройках созданной интеграции найдите поле "Redirect URI" или "URI перенаправления"
  2. Скопируйте URL из поля выше (или используйте автоматически сгенерированный):
  3. {{ redirect_uri_display }}
  4. Вставьте этот URL в настройки интеграции в AmoCRM
  5. Важно: URL должен точно совпадать (включая протокол https:// и завершающий слэш)

Шаг 3: Заполнение данных в форме выше

  1. Домен: ваш домен AmoCRM (например, kmrprofi.amocrm.ru)
  2. Client ID: скопируйте из настроек интеграции в AmoCRM
  3. Client Secret: скопируйте Secret key из настроек интеграции
  4. Redirect URI: должен совпадать с тем, что указан в AmoCRM (обычно заполняется автоматически)
  5. Нажмите "Сохранить"

Шаг 4: Авторизация через OAuth

  1. После сохранения настроек появится кнопка "Подключить (OAuth)"
  2. Нажмите на неё — откроется страница авторизации AmoCRM
  3. Войдите в ваш аккаунт AmoCRM (если не авторизованы)
  4. Разрешите доступ интеграции к данным вашего аккаунта
  5. После успешной авторизации вы будете перенаправлены обратно в настройки
  6. Статус изменится на подключено

⚠️ Важные замечания

  • OAuth токен имеет полные права доступа (включая список пользователей)
  • Access token действителен 24 часа, затем автоматически обновляется через Refresh token
  • Refresh token действителен 3 месяца, после чего потребуется повторная авторизация
  • Если используете long-lived token, OAuth не требуется (но могут быть ограничения прав)
  • После успешной авторизации можно перейти к миграции данных
