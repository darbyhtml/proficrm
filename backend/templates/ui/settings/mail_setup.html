Пересохранить пароль SMTP

Введите действующий пароль вашей учётки SMTP (smtp.bz / Яндекс / другой провайдер). Он будет зашифрован текущим MAILER_FERNET_KEY и сохранён в БД. Приёмник и сервер не трогаем — только пароль.

{% csrf_token %}
Что произойдёт после сохранения:
  1. Пароль зашифрован Fernet и записан в GlobalMailAccount.smtp_password_enc.
  2. Сразу проверим обратную расшифровку — результат должен совпасть с введённым.
  3. Событие записывается в журнал действий (audit) для compliance.
