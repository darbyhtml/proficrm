# Оценка проекта CRM ПРОФИ

## Общая информация
- **Тип проекта**: Полнофункциональная CRM система
- **Backend**: Django 6.0 (Python)
- **Frontend**: Django Templates + TailwindCSS
- **Mobile**: Android приложение (Kotlin)
- **База данных**: PostgreSQL (production), SQLite (dev)
- **Интеграции**: amoCRM API

---

## Детальная оценка по компонентам

### 1. Backend Core (Django Framework) - 150,000 ₽

**Компоненты:**
- Настройка Django проекта с production-ready конфигурацией
- Модели данных (Company, Contact, Task, User, Branch и др.)
- Система миграций (24+ миграций для companies, 4+ для других модулей)
- Настройка безопасности (CSRF, CORS, SSL)
- Middleware для rate limiting и безопасности

**Обоснование:** Базовая настройка Django с production конфигурацией, модели данных, миграции - это фундамент проекта. Стандартная стоимость: 80-120 часов × 1,500 ₽/час = 120,000-180,000 ₽. Средняя оценка: **150,000 ₽**
