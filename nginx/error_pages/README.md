# Кастомные страницы ошибок Nginx

Этот каталог содержит кастомные HTML страницы для обработки ошибок nginx (502, 503, 504, 500).

## Особенности

- **Забавный дизайн**: Каждая страница имеет уникальный дизайн с градиентами и анимациями
- **Автообновление**: Страницы автоматически обновляются каждые 5 секунд
- **Ручное обновление**: Кнопка для немедленного обновления
- **SVG иллюстрации**: Каждая страница содержит уникальную SVG картинку
- **Адаптивный дизайн**: Работает на всех устройствах

## Использование

Страницы автоматически монтируются в nginx контейнер через docker-compose.staging.yml.

Конфигурация nginx находится в `nginx/staging.conf` и настроена на использование этих страниц для ошибок 500, 502, 503, 504.

## Файлы

- `502.html` - Bad Gateway (технические шоколадки)
- `503.html` - Service Unavailable (техобслуживание)
- `504.html` - Gateway Timeout (таймаут)
- `500.html` - Internal Server Error (внутренняя ошибка)
