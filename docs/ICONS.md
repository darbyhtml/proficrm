# Наборы иконок в проекте

Чтобы не потерять единый стиль и быстро подбирать иконки в новых местах.

---

## Heroicons (Outline)

- **Источник:** [heroicons.com](https://heroicons.com/) → Outline (24×24)
- **Лицензия:** MIT (Tailwind Labs)
- **Стиль:** контурные, `stroke`, без заливки (`fill="none"`), `stroke-width="1.5"` или `2` для кнопок рядом с другими.

**Где используем:**
- Страница настроек `/settings/` — карточки разделов (Доступы, Пользователи, Филиалы, Справочники, Журнал действий, Импорт, amoCRM, Колонки компаний, Безопасность, Мобильное приложение, Статистика звонков, Лог ошибок).
- Настройки → Пользователи: иконка ключа (Heroicons key outline), кнопки «Редактировать» и «Выход» — тоже в одном стиле (stroke-width 2).

**Параметры SVG для единообразия:**
```html
<svg xmlns="http://www.w3.org/2000/svg" class="h-5 w-5" fill="none" viewBox="0 0 24 24" stroke="currentColor" stroke-width="1.5">
  <path stroke-linecap="round" stroke-linejoin="round" d="…" />
</svg>
```
Для кнопок-действий в одном ряду (как в таблице пользователей): `stroke-width="2"`, `width="16" height="16"`.

**Как брать иконки:** [Heroicons Outline](https://heroicons.com/) → нужная иконка → Copy SVG. При необходимости подравнять `stroke-width` под соседние элементы.

---

## Исключения (кастомные SVG)

- **Мобильное приложение** (карточка на `/settings/`): простой контур смартфона — `<rect>` + `<circle>` (кнопка), чтобы иконка не «поехала» в рендере.
- При необходимости можно добавлять минимальные кастомные SVG в том же стиле (viewBox 0 0 24 24, stroke, round caps).
