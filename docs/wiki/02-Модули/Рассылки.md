# Модуль: Рассылки (Mailer)

> Email-кампании, SMTP-менеджмент, очередь отправки.

## Возможности

- Email-кампании (plain text + HTML)
- Массовая отправка с фильтрацией получателей
- Состояния: draft → ready → sending → sent (+ paused, stopped)
- Отложенная отправка (send_at)
- Вложения файлов
- Шаблоны писем

## SMTP

| Тип | Модель | Описание |
|-----|--------|---------|
| Пользователя | `MailAccount` | Yandex, кастомный SMTP |
| Глобальный | `GlobalMailAccount` | smtp.bz (единый для CRM) |

- Пароли: **Fernet** шифрование
- Rate limiting: per-minute + per-day + per-user
- Квоты: синхронизация с smtp.bz API

## Очередь отправки

`CampaignQueue` — последовательная обработка:
- Приоритет + deferral при исчерпании квоты
- Circuit breaker для transient ошибок
- pending → processing → completed

## Отписки

- `Unsubscribe` — список отписанных
- `UnsubscribeToken` — токенизированные ссылки
- `EmailCooldown` — cooldown после отписки
