# Модуль: Задачи (TasksApp)

## Возможности

- CRUD задач с UUID PK
- Назначение исполнителя
- Статусы: new → in_progress → done / cancelled
- Повторяющиеся задачи (RFC 5545 RRULE: daily, weekly, monthly)
- Комментарии
- История событий (создание, смена статуса, назначение)
- Типы задач с иконками и цветами
- Пометка «срочно»

## Celery

| Задача | Интервал |
|--------|---------|
| `generate-recurring-tasks` | Ежедневно 06:00 MSK |

## Модели

- `TaskType` — типы (иконки, цвета)
- `Task` — задачи (UUID, RRULE)
- `TaskComment` — комментарии
- `TaskEvent` — история изменений
