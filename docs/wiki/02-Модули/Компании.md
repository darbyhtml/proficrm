# Модуль: Компании

> Ядро CRM. Управление компаниями, контактами, сделками, заметками.

## Возможности

- CRUD компаний с иерархией (родитель/дочерние)
- Контактные лица (email, телефон — множественные)
- Сделки/воронка (CompanyDeal)
- Заметки с вложениями
- Холодные звонки и отчётность
- Управление контрактами с напоминаниями об истечении
- Полная история изменений (CompanyHistoryEvent)
- Soft-delete с workflow подтверждения
- Привязка к филиалам (branch-based access control)

## Поиск

**PostgreSQL FTS + pg_trgm** (без внешних движков):
- Полнотекстовый по tsvector/tsquery
- Триграммный (similarity) для нечёткого поиска
- Модель `CompanySearchIndex` — отдельная таблица индекса
- Переиндексация: ежедневно в 00:00 MSK (`reindex-companies-daily`)
- Команда: `python manage.py rebuild_company_search_index`

## Справочники

- `CompanyStatus` — статусы (prospect, customer, partner...)
- `CompanySphere` — отрасли
- `Region` — регионы
- `ContractType` — типы контрактов
