# Модуль: Телефония (PhoneBridge)

## Возможности

- Регистрация телефонных устройств (FCM token)
- QR-спаривание (MobileAppQrToken)
- Входящие/исходящие вызовы (CallRequest)
- Телеметрия устройств (батарея, сигнал, связь)
- Пакетные логи вызовов (PhoneLogBundle)
- Управление версиями Android-приложения (MobileAppBuild)

## API

- `RegisterDeviceView` — регистрация устройства
- `QrTokenCreateView` / `QrTokenExchangeView` — QR-спаривание
- `PullCallView` — polling входящих вызовов
- `UpdateCallInfoView` — обновление статуса
- `DeviceHeartbeatView` — heartbeat
- `PhoneTelemetryView` — телеметрия
- `PhoneLogUploadView` — загрузка логов
