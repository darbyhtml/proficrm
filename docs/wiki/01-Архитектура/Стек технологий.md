## Поиск

PostgreSQL FTS + pg_trgm — без внешних поисковых движков.
- Полнотекстовый поиск (tsvector/tsquery)
- Триграммный поиск (similarity)
- Пороги: absolute_min=0.5, relative_factor=0.15, similarity=0.4
