# Структура проекта

## Django-приложения (14 штук)

```
backend/
├── accounts/      — пользователи, роли, филиалы, magic link
├── companies/     — компании, контакты, сделки, поиск (ядро CRM)
├── messenger/     — live-chat, виджет, оператор-панель, SSE
├── mailer/        — email-рассылки, SMTP, очередь
├── tasksapp/      — задачи, повторяющиеся, события
├── ui/            — фронтенд views, дашборд, настройки, отчёты
├── notifications/ — уведомления, напоминания, анонсы
├── phonebridge/   — телефония, мобильное приложение
├── audit/         — логи действий, ошибки
├── policy/        — правила, конфиги политик
├── amocrm/        — интеграция AmoCRM, миграция
├── crm/           — ядро Django (settings, urls, middleware)
├── core/          — утилиты
└── templates/     — 89 HTML шаблонов
```

## Крупнейшие модули (по LOC)

1. **ui/views.py** — ~13,500 строк (фронтенд логика)
2. **messenger/** — ~1,129 строк моделей + API + widget
3. **companies/** — ~818 строк моделей

## Frontend-сборка

```
frontend/src/main.css → (Tailwind build) → backend/static/ui/css/main.css
```

Команда: `npm run build:css`

## Роли пользователей

| Роль | Scope | Описание |
|------|-------|---------|
| admin | GLOBAL | Полный доступ ко всему |
| branch_director | BRANCH | Управление своим филиалом |
| sales_head | BRANCH | Управление отделом продаж |
| group_manager | BRANCH | Управление группой менеджеров |
| manager | SELF | Только свои компании |
