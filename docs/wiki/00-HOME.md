# CRM ПРОФИ — Живая Документация

> Система «Живой Памяти» Claude Code + Obsidian.
> Эта документация автоматически обновляется Claude Code при каждом изменении проекта.

---

## Навигация

### Архитектура
- [[Стек технологий]] — Python 3.13, Django 6, PostgreSQL 16, Redis 7, Celery
- [[Структура проекта]] — Django-приложения, файловая структура, 439 файлов Python
- [[База данных]] — 55 моделей, связи, миграции
- [[API]] — REST API, 50+ эндпоинтов, OpenAPI/Swagger

### Модули
- [[Компании]] — ядро CRM: компании, контакты, сделки, поиск
- [[Мессенджер]] — live-chat виджет, оператор-панель, SSE, каналы
- [[Рассылки]] — email-кампании, SMTP, очередь отправки
- [[Задачи]] — задачи, повторяющиеся (RRULE), события
- [[Уведомления]] — in-app, push, напоминания по контрактам
- [[Телефония]] — PhoneBridge, мобильное приложение, QR-спаривание
- [[Аудит и политики]] — логирование действий, ошибки, правила

### Инфраструктура
- [[Docker и сервисы]] — контейнеры, docker-compose, volumes
- [[Nginx]] — reverse proxy, SSL, CORS, SSE
- [[Деплой workflow]] — staging → тестирование → прод (вручную)
- [[Celery задачи]] — периодические задачи, расписание

### Статус проекта
- [[Статус прод]] — что задеплоено, текущая версия
- [[Статус staging]] — что тестируется, текущие эксперименты
- [[Известные проблемы]] — баги, TODO, технический долг

### Журнал
- [[Changelog]] — история изменений по датам
- [[Архитектурные решения]] — ключевые решения и их обоснования
