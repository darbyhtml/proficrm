# Roadmap

## Приоритет 1 — Мессенджер до продакшена

**Цель:** Довести live-chat до уровня JivoChat/Chatwoot — надёжный SaaS продукт.

- [x] Real-time доставка сообщений (SSE) — **РЕШЕНО**: тройная дедупликация в widget.js + host nginx proxy_buffering
- [ ] Typing-индикаторы в обе стороны
- [ ] Кампании (автоприглашения посетителям)
- [ ] Browser push-уведомления для операторов
- [ ] Оценка диалога (rating) — stars + NPS
- [ ] Автоматизация: приветствие, эскалация, auto-assign
- [ ] Нагрузочное тестирование (несколько одновременных виджетов)
- [ ] Деплой на прод после полного QA

## Приоритет 4 — Улучшения

- [ ] Analytics dashboard для мессенджера (время ответа, CSAT, нагрузка)
- [ ] Мобильное приложение для операторов (push + ответ из notification)
- [ ] Интеграция с AmoCRM (синхронизация контактов)
- [ ] Bulk-операции с компаниями (массовое назначение, статус)
- [ ] Export/import компаний (CSV/Excel) — улучшение текущего

- Knowledge base для виджета (FAQ до оператора)

- Webhook-уведомления для внешних интеграций
- API rate limiting dashboard для мониторинга
